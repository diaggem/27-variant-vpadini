# Вариант 27 — Океанические впадины

## Условие

Программа работает с данными о 20 океанических впадинах. Для каждой впадины указаны название, глубина, океан и тип (желоб, хребет или разломная).

Что делает программа:
- Считает впадины глубже 6000 метров
- Отбирает только желоба и сортирует их по убыванию глубины
- Показывает океаны трёх самых глубоких впадин
- Меняет данные впадины по её глубине
- Ищет все впадины в заданном океане

## Запуск

Скомпилировать и запустить любой компилятором C++.

## Пример работы
6000м: 16

Желобы (по убыванию глубины):
Марианская, 11034 м, Тихий, желоб
Тонга, 10882 м, Тихий, желоб
Филиппинская, 10540 м, Тихий, желоб
...

Океаны 3 глубочайших:

Марианская - Тихий

Тонга - Тихий

Филиппинская - Тихий

Океан: Тихий

Результат:
Марианская, 11034 м, Тихий, желоб
Тонга, 10882 м, Тихий, желоб
...


## Сборка

Никаких внешних библиотек не требуется. Нужен только стандартный C++ (iostream, cstring).
